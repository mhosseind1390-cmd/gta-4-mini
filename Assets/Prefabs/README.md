# Prefabs Directory

This directory contains all game prefabs:

- **Player.prefab**: Player character
- **Car.prefab**: Drivable vehicle
- **Enemy.prefab**: AI enemy
- **Police.prefab**: Police officer NPC
- **Weapon.prefab**: Pickup weapon
- **Pickup.prefab**: Money/item pickup

## Instructions

1. Create these prefabs in Unity
2. Assign scripts to appropriate GameObjects
3. Configure Rigidbodies and Colliders
4. Set up animations
