# Scenes Directory

This directory contains game scenes:

- **MainScene.unity**: Main gameplay scene
- **MenuScene.unity**: Main menu (optional)
- **LoadingScene.unity**: Loading screen (optional)

## Setting up MainScene

1. Create a new 3D scene
2. Add ground plane
3. Add GameManager, UIManager, PoliceManager
4. Add Player prefab
5. Add multiple Car prefabs
6. Add Enemy and Police prefabs
7. Set up Canvas and UI elements
8. Save as MainScene.unity
